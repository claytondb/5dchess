# Changelog

All notable changes to 5D Chess: Multiverse Tactics.

## [0.3.0] - 2026-02-25

### Fixed
- PWA manifest now uses SVG icon instead of missing PNG files
- Icons work correctly for mobile installation

## [0.2.0] - 2026-02-21

### Added
- **PWA Support** - Installable on mobile devices
- Improved meta tags for better mobile experience

## [0.1.0] - 2026-02-18

### Added
- **Core Gameplay** - Real-time pausable tactical combat
- **Temporal Jumps** - Send ships back in time to create timeline branches
- **Multi-Timeline Combat** - Manage up to 3 concurrent timelines
- **Tutorial Level** - Learn the basics of multiverse combat
- **Main Menu** - Title screen with game options
- **Ship Types** - Fighter and Cruiser (flagship)
- **AI Opponent** - Basic enemy fleet behavior
- **Portrait Mode** - Mobile-friendly layout
- **Touch Controls** - Full mobile support

### Technical
- Phaser 3 game framework
- Vanilla JavaScript
- HTML5 Canvas rendering
- GitHub Pages deployment

### Documentation
- Full Requirements document
- Game Design Document (GDD)
- UX Design specifications
